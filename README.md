# Maple Converter
Desktop application for converting text between several encodings.
<div>
<img src="/doc/window.png" width="500px"</img> 
</div>

## Features
- ASCII to HEX 
- HEX to ASCII 
- ASCII to EBCDIC
- EBCDIC to ASCII

## Requirements
- JDK 8+
